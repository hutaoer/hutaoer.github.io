# 原生App和Flutter通信接口文档

# 原理

  • 原生和Flutter之间通过BasicMessageChannel实现双向通讯，双向通讯的数据为字符串类型。 通讯的内容为json结构：
{ "code":"接口名称",
  "data":"接口请求数据或返回数据"
}

原生和Flutter通过code来识别请求和返回的接口，通过data获取请求或返回的数据

注册的channel名称：basicChannel
