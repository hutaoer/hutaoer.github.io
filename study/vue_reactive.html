    # Vue响应式原理

    # Vue 2.0

    • Vue 采用数据劫持结合发布者-订阅者模式的方式来实现数据的响应式。Object.defineProperty来劫持数据的setter，getter，在数据变动时发布消息给订阅者，订阅者收到消息后进行相应的处理。
